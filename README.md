# Real-Debrid API

A simple API interface for real-debrid.com

## Getting started

### Install

You may install the Real-Debrid API with Composer (recommended) or manually.

### System Requirements

You need **PHP >= 5.3.0**

## API Reference

### User

```
$user = new \RealDebrid\User();
```

#### Login

Login to real-debrid.com.

```
$user->login('username', 'password');
````
